# 🛒 Product Category Viewer

A responsive web application built using **React + Vite** that displays products by category with pagination support.  
Users can browse products, load more items, and view category-wise product listings.

---

## 🚀 Features

- 📦 Display products by category  
- 🔄 Pagination / Load More functionality  
- ⏳ Loading indicators for better user experience  
- ❌ Error handling for failed API requests  
- ⚡ Fast performance using React Hooks  
- 📱 Responsive UI design  

---

## 🛠️ Tech Stack

- **Frontend:** React (JavaScript)
- **Build Tool:** Vite
- **Styling:** CSS(TailwindCSS)
- **State Management:** React Hooks (`useState`, `useEffect`, `useCallback`)
- **API Handling:** Fetch / Async Functions
